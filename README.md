# glove-py

Practice: Python implementation of GloVe word embedding algorithm

## Dependencies

- Python3.6
- [Poetry](https://github.com/sdispater/poetry)

## Installation

```bash
cd glove-py
poetry install
```

## Usage

```bash
cd glove-py
poetry run python main.py
```
